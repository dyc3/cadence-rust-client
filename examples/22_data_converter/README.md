# Example 22: Data Converter

## Overview

This example demonstrates custom data converters for serialization.

## Features Demonstrated

- **Custom DataConverter**: Implementing custom serialization
- **Alternative Formats**: Using Protobuf, MessagePack, etc.
- **Encryption**: Encrypting data at rest

## Running the Example

```bash
cargo run -p data_converter
```
